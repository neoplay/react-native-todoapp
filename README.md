# react-native-todoapp
Simple React Native Todo App with Redux

## Dependencies
- React (16.0.0 Alpha 12)
- React Native (0.45.1)
- Redux (3.7.0)
- Redux Thunk (2.2.0)
- React Redux (5.0.5)
- React Native Elements (0.13.0)
- React Native Vector Icons (4.2.0)
